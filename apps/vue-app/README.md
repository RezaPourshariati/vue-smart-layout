# @adaptive-auth/vue-app

Vue 3 + Vite SPA for AdaptiveAuth.

## Commands

From repo root:

```sh
pnpm dev:vue-app
pnpm build:vue-app
pnpm type-check:vue-app
pnpm test:vue-app
pnpm test:e2e
```

From this directory:

```sh
pnpm dev
pnpm build
pnpm type-check
```

## Environment

Copy `.env.example` to `.env` and set `VITE_API_BASE_URL` (default points at the local auth server).
